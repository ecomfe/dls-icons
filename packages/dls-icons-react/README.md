# dls-icons-react

## Installation

```shell
npm i -D dls-icons-react
```

## Usage

```jsx
import { IconArrowRight } from "dls-icons-react";

export default function Title () {
  return (
    <div className="title">
      <h1>
        Hello Icons
        <IconArrowRight style={color: '#999', marginLeft: 5}/>
      </h1>
    </div>
  )
}
```
