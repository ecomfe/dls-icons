# dls-icons-data

![](https://img.shields.io/npm/v/dls-icons-data.svg)

## Installation

```shell
npm i -D dls-icons-data
```

## Usage

```js
import { dataArrowRight } from 'dls-icons-data'

/*
{
  {
    name: 'icon-arrow-right',
    content: '<path d="..." ...',
    width: 24,
    height: 24
  }
}
*/
console.log(dataArrowRight)
```
